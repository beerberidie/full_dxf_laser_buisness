‚ÑπÔ∏è About Presets

Machine Settings Presets allow you to save commonly used laser cutting parameters for specific material types and thicknesses.

Benefits:

  • Faster laser run logging - select a preset instead of entering all settings manually
  • Consistency - ensure the same settings are used for the same materials
  • Quality control - standardize cutting parameters across operators
  • Easy updates - change settings in one place and they apply to all future runs

Usage:

  • When logging a laser run, select a preset from the dropdown
  • The preset dropdown automatically filters by material type and thickness
  • Only active presets appear in the dropdown
  • Deactivate presets you no longer use instead of deleting them (preserves history)
